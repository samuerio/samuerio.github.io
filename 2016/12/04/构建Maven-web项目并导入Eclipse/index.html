构建Maven web项目并导入Eclipse

issUn 2016-12-04

使用Maven Archetype插件

直接使用Maven Archetype Plugin进行web项目骨架的生成

Eclipse导入时选Existing Maven Projects即可，Eclipse就能将其识别为Maven Web项目。最后调整项目的Build Path,由默认的mavenweb-test/target/classes调整为mavenweb-test/src/main/webapp/WEB-INF/classes就大工完成了。

手动构建Maven Web项目

创建项目文件夹，并添加pom.xml文件

创建Maven Web的src/main/java 、src/main/resources、src/test/java、src/test/resources、src/main/webapp等目录

导入Eclipse

Eclipse导入时选Existing Maven Projects。以这种方式构建的Maven Web项目导入时Eclipse不会自动将其识别为Web项目，需要改其Eclipse的配置文件，好让IDE识别。

1.编辑.project文件

找到<natures></natures>添加上,并刷新项目

2.编辑项目的Project Facets

打开项目properties，找到Project Facets,并勾选上右侧的Dynamic Web Module和Java

并点击底部的Further configuration available...进行WebContent变量的配置(默认的为WebRoot将其调整为webapp路径即可)

最后同上面一样进行Build Path调整即可。
